#User management

When a client send a message or ask about a product, this will be registered into Database as customer.

Create the customer and store ID into a cookie

That data will be used in future asks about products. Data will be used only by Admin and same user
App will generate a short-time access token (5 minutes) to request the info about client.
Data will be retrieve if token is validate
Data will be store in Customer context

Add on metadata total users

  name: String
  email: String
  phone: String
  firstTime: Boolean
